# sql2json
Cette commande, écrit en Pascal (Turbo Pascal ou Free Pascal), permet de convertir un fichier SQL en JSON.

<h3>Syntaxe</h3>

<b>SQL2JSON</b> <i>source</i>.SQL

<h3>Paramètres</h3>

<table>
  <tr>
    <th>Nom</th>
    <th>Description</th>
  </tr>
  <tr>
    <td><i>source</i></td>
    <td>Ce paramètre permet d'indiquer le fichier source SQL à convertir.</td>
  </tr>
</table>
